安装二进制包#

1.下载相应go版本并解压

rm -rf /usr/local/go && tar -C /usr/local -xzf go1.14.3.linux-amd64.tar.gz

如果已经安装会先删除go目录

2.添加环境变量

echo "export PATH=$PATH:/usr/local/go/bin" >> /etc/profile && source /etc/profile

3.检查是否安装成功

go version
