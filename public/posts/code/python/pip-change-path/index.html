修改pip下载位置#

之前装python3下了很多包,看着日渐填满的C盘空间就想给 pip 下载的包挪个位置!

搜到的改包路径其实就是修改 site.py 文件的两个参数:

具体步骤#

1.查找文件所在位置#

#所有依赖的位置
python -m site

#查看 site.py 位置
python -m site -help

2.找到文件之后修改以下参数内容即可#


USER_BASE =  #包的位置

USER_SITE =  #pip脚本位置

之后就ok了
