MongoDB#

c++开发的 文档数据库(键值对)

win使用#

添加环境变量C:\Program Files\MongoDB\Server\5.0\bin

mongodb服务#

安装mongodb服务

mongod.exe --config "C:\Program Files\MongoDB\Server\5.0\bin\mongod.cfg" --install

mongod.exe --remove
# 移除服务

默认安装服务并启动会因为权限原因无法启动

解决方法

# 管理员方式启动powershell
sc delete MongoDB
# 或者试试 mongod.exe --remove 删除服务
mongod --install -f "C:\Program Files\MongoDB\Server\5.0\bin\mongod.cfg"
net start mongodb

此后即可通过服务启动mongoBD

命令行运行mongodb服务器#

cd C:\
md "\data\db"
# 创建数据目录

mongod.exe --dbpath="c:\data\db"
mongo

进入mongo之后运行的是一个javascript shell
