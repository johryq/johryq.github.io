Windows10工具#

记录一些在用的软件

大部分软件为Github地址

系统工具#

  • Wow : 快速查找电脑文件

  • PowerToys:微软出品,工具集合:颜色选择器,窗口布局,资源管理器预览,图像调整,键盘管理,重命名,快速查找文件,快捷键指南,视频会议经验

  • Quicklook : 只需按下空格键即可查看文件内容,快速(也可微软商店搜索下载)

  • EarTrumpet : win10分别控制各个程序的音量(微软商店搜索下载)

  • Everything : 快速搜索文件配合 Wow 使用

  • 7 taskbar tweaker :

    • 高级选项 –> no_width_limit –> 1
    • 即可修改任务栏宽度

  • TranslucentTB | github地址 : 透明任务栏

  • listen1: 聚合音乐平台

Shell#

  • MobaXterm : xshell与xftp结合版

  • windows Terminal : 炫酷方便的shell(微软商店下载)

  • Xshell : 以前使用最多的shell
