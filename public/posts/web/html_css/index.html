定位#

定位 = 定位模式 + 边偏移

定位模式#

position: static ｜relative ｜sbsolute

  • static
    • 类似标准流
  • relative
    • 相对元素本身原来位置偏移
    • 原来位置继续保存，不脱标
  • absolutle
    • 一般相对父元素移动（父元素需加绝对定位），没有则以浏览器
    • 如父没有，则以上级有定位的元素移动
  • fixed
    • 固定定位
    • 浏览器可视窗口为定位点
    • 脱标，不占有标准流
  • sticky
    • 粘性定位
    • 占有原来位置
    • 相对于绝对定位的整合（以窗口可视区为准）
    • 必加边偏移

子绝父相
