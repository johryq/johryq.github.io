sudo#

使普通用户执行命令拥有root权限

Linux user add sudo authority#

方法一:修改 sudoers#

1.未安装 sudo 需要登录 root 执行:

apt install sudo #ubuntu
yum install sudo #centos

2.设置( root 下)

vi /etc/sudoers

3.添加如下内容到配置文件

用户名 ALL=(ALL) ALL
用户名 ALL=(ALL) NOPASSWD: ALL  #执行sudo时不用输入密码

# or 
sudo usermod -aG sudo [name-of-user]

最后强制写入退出

方法二:usermod添加进sudo 组#

usermod -aG sudo 用户名

sudo相关命令#

# 版本及相关信息
sudo -V

# 当前用不sudo权限
sudo -l
