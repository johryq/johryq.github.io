# NEST

## start

```bash
nest new [projectName]

nest new ict [name]

# create CRUD modele
nest g resource [name]
# 不生成测试文件
nest g resource [name] --no-spec

# create servicce
nest g service [servicesName]
nest g controller [controllerName]
nest g module [moduleName]

```
